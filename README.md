# Diabetes Dataset — Exploratory Data Analysis

An exploratory data analysis (EDA) of the Pima Indians Diabetes Dataset 
using Python.

## Dataset
- 768 entries, 9 features
- Source: Kaggle (Pima Indians Diabetes Database)

## What I explored
- Distribution of glucose levels across the population
- Correlation between all features using a heatmap
- Relationship between Age and BMI, colored by diabetes outcome
- Comparison of key features (Glucose, BMI, Insulin, Age) between 
  diabetic and non-diabetic groups using box plots

## Key finding
Glucose level is the strongest predictor of diabetes in this dataset, 
followed by BMI and Age.

## Tools
Python · pandas · NumPy · matplotlib · seaborn · Jupyter Notebook
